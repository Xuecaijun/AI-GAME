# 比赛当天冲刺清单

## 赛前准备

- 准备至少 1 个可用的大模型 API Key
- 准备 3 份演示简历文本
- 准备 4 个原创面试官头像或角色卡
- 准备 1 套主视觉配色和标题文案
- 准备 1 套首页标题和结果页截图文案

## 比赛当天优先级

1. 先跑通流程，不要先沉迷特效。
2. 先确认 API 正常，再追求问题质量。
3. 先保证结果页有说服力，再补成就和隐藏剧情。

## 演示建议

- 第一局用 AI 生成简历，展示游戏感。
- 第二局用真实简历，展示实用感。
- 刻意选一个高难度，展示追问压迫感。

## 风险兜底

- API 挂了就切 Mock 模式
- 模型输出乱了就退回本地追问
- 时间不够就砍隐藏玩法，只保留 4 轮主线
